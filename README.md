# Projeto de Aplicação de Modelos Lineares Generalizados em Dados Cinematográficos

## Introdução

Este projeto visa aplicar Modelos Lineares Generalizados (MLG) a um conjunto de dados relacionados a filmes cinematográficos. Utilizaremos técnicas estatísticas avançadas para compreender e modelar as relações entre diversas variáveis e os resultados de desempenho dos filmes.

## Objetivos

1. **Exploração de Dados:** Realizar uma análise exploratória completa do conjunto de dados, identificando padrões, distribuições e possíveis relações entre as variáveis.
2. **Pré-processamento de Dados:** Realizar limpeza e transformação dos dados, tratando valores ausentes, outliers e convertendo variáveis necessárias para o formato adequado.
3. **Modelagem Estatística:** Aplicar Modelos Lineares Generalizados para entender as relações entre as variáveis independentes e a variável resposta. Avaliar diferentes distribuições de erro e funções de ligação.
4. **Avaliação do Modelo:** Utilizar métricas apropriadas para avaliar o desempenho dos modelos, como R², deviance, e realizar validação cruzada para verificar a robustez dos resultados.
5. **Interpretação dos Resultados:** Interpretar os parâmetros estimados, identificar variáveis significativas e comunicar os insights obtidos.

## Como Usar

**Requisitos:**
- R (~4.2.2)

**Instruções:**
1. Clone este repositório: `git clone https://github.com/muq33/trabalho_mlg.git`
2. Navegue até o diretório do projeto: `cd trabalho_mlg`
3. Abra o projeto: `Trabalho.rproj`
